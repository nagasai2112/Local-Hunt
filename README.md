Local Hunt is a location-based web application that helps users discover nearby vendors, small shops, and local businesses. Users can search for trusted vendors in their area, view shop details, chat with vendors, and navigate to the shop location. Vendors can register, list services/products, and manage their business profile on the platform.

## üìå Abstract (Project Summary)

Local Hunt is a platform that connects customers with nearby street shops, vendors, and small businesses using location-based services. It helps users discover trusted vendors in their city while allowing vendors to promote their business, update details, and interact with customers. The system supports secure login, session management, navigation, reviews, messaging, vendor recommendations, and a personalized shopping experience.  
The goal is to support local businesses and make discovery simple for users.

## ‚≠êDescription

Local Hunt is a MERN + Firebase location-based web application that helps users discover nearby vendors and small businesses.
